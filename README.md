# BeckPro Tool

BeckPro Tool is a professional unlocking tool for Xiaomi devices. This tool allows users to unlock bootloaders, perform FRP (Factory Reset Protection) bypass, retrieve device information, and much more.

## Features
- FRP Bypass
- Bootloader Unlocking
- Device Information Retrieval
- Anti-Crack Detection
- Backup and Restore Operations

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/jurabeck8688-tech/beckpro-tool.git
   ```
2. Navigate to the repository directory:
   ```bash
   cd beckpro-tool
   ```
3. Run the main Python script:
   ```bash
   python main.py
   ```
